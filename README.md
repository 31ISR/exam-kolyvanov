# ТЕХНИЧЕСКОЕ ЗАДАНИЕ НА ЭКЗАМЕН
## Редизайн сайта питомника "Сад СибНИИСХоза"

**Сайт для редизайна:** https://садсибниисхоза.рф

---

## ОПИСАНИЕ ПРОЕКТА

Заказчик обратился к вам с задачей выполнить редизайн сайта плодопитомника СибНИИСХоза (Омск). Питомник занимается продажей саженцев плодово-ягодных и декоративных культур с открытой и закрытой корневой системой.

**Целевая аудитория:**
- Садоводы-любители (30-70 лет)
- Владельцы частных домов и дач
- Ландшафтные дизайнеры
- Фермеры и сельхозпроизводители

---

## ОБЯЗАТЕЛЬНЫЕ ТРЕБОВАНИЯ

### 1. СТРУКТУРА САЙТА

**Навигация:**
- Шапка сверху вьюпорта сайта
- Логотип организации
- Кнопки "Контакты", "Гостевая книга", "Цены", "Вход" и "Регистрация"
- Поп-ап окна входа (необходимо оформить) и регистрации (просто пустая форма)
- Должен скрываться при скролле вниз и появляться при скролле вверх
- Превращаться в бургер меню на мобильной версии, где будет раскрываться меню с кнопками перехода и авторизации

**Футер:**
- Контактная информация
- Название
- Копирайт

**Главная страница должна содержать:**
- Навигация
- Главный баннер-слайдер/герой-секция
- Форма поиска
- Каталог категорий растений (текущая вкладка "описание сортов" в навигации)
- Информация о питомнике (3-4 блока)
- Футер

**Страница растения (голубика "Река"):**
- Навигация
- Переход на страницу из каталога категорий растений на главной странице
- Название
- Описание
- Изображение
- Футер

---

### 3. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ

**HTML:**
- Валидная семантическая разметка
- Атрибуты доступности
- Правильная структура текста

**CSS:**
- Адаптивная вёрстка
- CSS переменные
- Hover, transitions, animation
- Современные методы CSS

**Адаптивность:**
- Все страницы должны корректно отображаться на мобильных устройствах
- Мобильное меню для навигации
- Читаемые размеры шрифтов на всех устройствах

---

### 4. ДИЗАЙН ТРЕБОВАНИЯ

**Обязательные элементы:**
- Единая цветовая схема (2-3 основных цвета + акцентный)
- Использование логотипа
- Использование иконок
- Типографическая иерархия
- Использование шрифтовых пар

**Атмосфера:**
- Природная, "садовая" тематика
- Доверие и экологичность
- Понятно для возрастной аудитории
- "Легкий", неперегруженный дизайн
